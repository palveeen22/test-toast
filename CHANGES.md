# Изменения

## Рефакторинг контекста тостов

- Разделён монолитный `ToastContext.tsx` на три модуля:
  - `ToastContext.ts` — создание контекста
  - `ToastProvider.tsx` — провайдер с логикой управления тостами
  - `useToast.ts` — хук для доступа к контексту

## Реализация ToastProvider

- Добавление тостов через `addToast` с автогенерацией ID
- Дедупликация: повторный тост с тем же сообщением и типом сбрасывает таймер вместо создания дубликата
- Удаление тостов через `removeToast`

## Улучшение ToastItem

- Автоматическое скрытие тоста по таймеру (по умолчанию 3 сек)
- Пауза таймера при наведении мыши, возобновление при уходе
- Анимация появления (slide-in) и исчезновения (slide-out)
- Сброс таймера при повторном вызове того же тоста (`resetKey`)

## Подключение в App.tsx

- Раскомментирован и подключён `ToastProvider` и `useToast`
- Кнопки Success, Error, Warning теперь вызывают тосты

## Тестирование

- Добавлены зависимости: `vitest`, `jsdom`, `@testing-library/react`, `@testing-library/dom`, `@testing-library/jest-dom`
- Настроен Vitest в `vite.config.ts` с окружением `jsdom`
- Добавлены тесты в `src/test/toast.test.tsx`
